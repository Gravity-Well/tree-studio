Embed
Two lines of HTML.
Any site.

Drop a live, interactive tree into any webpage. No build step, no SDK, no dependencies.

<!-- Embed a shared tree --> <iframe src="https://treestudio.nusoftva.com/app ?embed=1 &view=userId/my-org-chart" width="100%" height="500" frameborder="0" ></iframe>
🔗

Share a link, get an embed

Every saved tree gets a shareable URL. Add ?embed=1 and it's an iframe-ready widget.

👁️

Read-only by default

Embedded views are view-only. Visitors can explore, zoom, and collapse — but cannot edit your tree.

🎨

Fits any layout

Set any width and height. The canvas pans and zooms within the frame. Works in Notion, Confluence, any CMS.

⚡

Always current

Edit your tree in Tree Studio and the embed updates everywhere it's used — no re-publishing needed.

🧮

Proprietary layout engine

Not a D3 wrapper. Tree Studio uses a custom tidy-tree algorithm inspired by Reingold-Tilford, Walker, and Buchheim — producing compact, readable layouts at any depth.
